# Museo

Aplicación fullstack para un museo.

## Descripción

Sistema web para gestión o presentación de un museo. El contexto exacto requiere ingestión profunda del código.

## Stack

- **Backend**: Node.js (`backend/` con `package.json`)
- **Frontend**: Angular 17.x (`frontend/` con `angular.json`, CLI v17.3.17)
