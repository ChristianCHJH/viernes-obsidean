# Metropolitano — Sistema de Monitoreo de Flota

Sistema web para monitorear buses, reportes de viaje y seguimiento GPS en tiempo real. Fue una **prueba técnica**.

## Descripción

Sistema de monitoreo de flota para el servicio de transporte Metropolitano. Permite rastrear buses en tiempo real mediante GPS, generar reportes de viaje y hacer seguimiento de la operación.

## Stack

- **Backend**: Node.js (`backend/` con `package.json`)
- **Frontend**: Angular (`frontend/` con `angular.json`)
